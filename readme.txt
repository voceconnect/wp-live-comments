Live comment loading and AJAX comment form written using Backbone.js.

== Installation ==

1. Upload `wp-live-comments` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Ensure your comments template will always include `<ol class="commentlist"></ol>`
1. Remove the call to `wp_list_comments()` from your comments template
